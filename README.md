# 🍔 B7.Burger

## 📖 Sobre o Projeto

O **B7.Burger** é uma landing page desenvolvida para representar uma hamburgueria moderna, com foco na apresentação visual dos produtos e na experiência do usuário.

O projeto foi construído utilizando apenas **HTML5** e **CSS3**, sem o uso de frameworks como Bootstrap ou bibliotecas JavaScript. O objetivo principal foi praticar a criação de interfaces web, organização de layouts e estilização de componentes de forma totalmente manual.

---

## 🚀 Funcionalidades

* Banner principal com destaque para a marca.
* Exibição de hambúrgueres em formato de cards.
* Campo de seleção para categorias de produtos.
* Layout moderno inspirado em aplicativos de delivery.
* Estrutura organizada e semântica utilizando HTML5.
* Estilização personalizada utilizando CSS3.

---

## 🛠️ Tecnologias Utilizadas

* CSS3
